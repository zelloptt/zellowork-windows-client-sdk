ZelloWork Client SDK Documentation

Zello for Work Client SDK for Windows allows to integrate your Windows application or web-page with full-featured Zello for Work client application. Key part of this SDK is ActiveX control which is as functional as standalone Zello for Work client application. You can use Visual C++, Visual Basic, Visual C# or any other programming language supporting hosting of ActiveX controls. For information on Zello server API please check http://zello.com/api/.
